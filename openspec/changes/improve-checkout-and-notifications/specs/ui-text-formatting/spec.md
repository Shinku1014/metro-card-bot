## ADDED Requirements

### Requirement: UI 文案优化
系统 SHALL 统一更新全系统的优惠券显示格式，将 `-2` 系列文字更改为 `减二`，并在冒号后加入空格。
- `五折:x` -> `五折: x`
- `-2:x` -> `减二: x`

#### Scenario: 主菜单卡片列表展示
- **WHEN** 用户点击 `/start` 或查看卡片列表
- **THEN** 卡片按钮上的文本应显示为 `(五折: 10 减二: 5)` 而不是旧格式

#### Scenario: 进站/出站通知
- **WHEN** 用户消耗优惠券后的通知消息
- **THEN** 系统 SHALL 显示包含新名称 `减二` 的提示语
